# Linux安装
- git clone https://github.com/google/googletest.git
- cd googletest-master  
创建文件夹mydir，用于作为cmake的目录  
- mkdir build  
进入build目录  
- cd build
输入cmake进行编译  
- cmake ..  
安装  
- make  
- sudo make install
